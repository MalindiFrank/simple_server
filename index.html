Node.js Simple Mock Server

This API allows you to manage:

  • /movies - Manage movies (title, year)
  • /series - Manage series (title, seasons)
  • /songs - Manage songs (title, artist)

Available Methods:

  • GET - Fetch all items
  • POST - Add an item (JSON body)
  • PUT - Update an item (by ID)
  • DELETE - Remove an item (by ID)

Sample JSON for Movies:

{
  "id": 1,
  "title": "Inception",
  "director": "Christopher Nolan",
  "year": 2010
}
